# Animal Based Diet Chatbot - Frontend

A React chat interface for the animal-based diet assistant.

## Setup

```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173` by default.

## Configuration

Create a `.env` file to configure the API backend URL:

```
VITE_API_URL=http://localhost:8000
```

## Tech Stack

- React 19
- Vite 8
- Vanilla CSS
